# 🚦 Road Accident Analysis Dashboard – Excel Project

## 📌 Project Overview

The **Road Accident Analysis Dashboard** is an interactive data analytics project built entirely in **Microsoft Excel** to analyze accident patterns, casualty trends, and contributing factors behind road accidents.

The purpose of this dashboard is to transform raw accident data into meaningful insights that can support **road safety planning, transport authorities, and data-driven decision-making**. The dashboard combines multiple visualizations, KPIs, and interactive filters to help users quickly explore accident trends and identify high-risk areas.

---

## 🎯 Project Objectives

The main objectives of this project were to:

* Analyze road accident severity and casualty distribution
* Identify high-risk vehicle categories
* Compare accident trends over time
* Understand the impact of road conditions and environment on accidents
* Build an interactive dashboard for quick decision-making

---

## 🔍 Key Dashboard Features

### 📊 Casualty Overview

* Displays **648,780 total casualties**
* Breakdown of:

  * Fatal casualties
  * Serious casualties
  * Slight casualties

### 🚗 Vehicle-wise Casualty Analysis

* Accident contribution by:

  * Cars
  * Motorcycles
  * Buses
  * Trucks
  * Other vehicle categories

### 📈 Monthly Trend Analysis (CY vs PY)

* Comparative trend visualization between **Current Year (CY)** and **Previous Year (PY)**
* Helps identify seasonal accident patterns and yearly changes

### 🛣️ Road Type Analysis

* Examines accident severity across different road types
* Highlights that **Single Carriageways contributed the highest number of casualties (479K+)**

### 🌧️ Road Surface Condition Insights

* Analysis of accident distribution based on:

  * Dry roads
  * Wet roads
  * Snow/Ice conditions
  * Flooded roads

### 🏙️ Urban vs Rural Accident Breakdown

* **Urban Areas:** 61% of casualties
* **Rural Areas:** 39% of casualties

### ☀️ Daylight vs Darkness Comparison

* Visual comparison of accident occurrence during:

  * Daylight
  * Darkness

### 🎛️ Interactive Filters (Slicers)

Users can dynamically filter data by:

* Month
* Weather Conditions
* Area Type
* Year
